# 安装



```
pip3 install chromadb 
```





## 验证



在 Python 环境中输入以下命令来验证安装是否成功：

```python
import chromadb
```





## Docker 安装



```
docker run -d --name chromadb-container -p 8899:8000 chromadb/chroma
```
