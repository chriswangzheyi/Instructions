# 涌现能力



**涌现能力（Emergent Ability）** 是指——
 当模型规模（参数量、训练数据、算力）**大到某个临界点**后，
 模型会**突然出现一些原本小模型完全不会的能力**，
 而且这些能力**不是显式训练出来的**。



| 模型规模      | 表现                               | 能力变化       |
| ------------- | ---------------------------------- | -------------- |
| 10M 参数      | 胡乱生成句子                       | 无语法、无逻辑 |
| 100M 参数     | 基本语法通顺                       | 会背模板       |
| 1B 参数       | 会回答常见问答                     | 但逻辑浅       |
| **10B+ 参数** | 🤯 能写代码、推理、理解笑话、做翻译 | 出现“涌现能力” |





## 💡 是什么原因造成的

### 1. **非线性叠加**

神经网络中的参数交互是高度非线性的，
 当规模大到一定程度，神经网络能**自己学会更复杂的特征表示**。

### 2. **信息容量突破**

大模型可以储存、组合更多知识，
 这就像人脑神经元多了之后，能组合出更多思维路径。

### 3. **数据分布更广**

大模型看到的文本更多样，
 所以能**自动学到“跨任务”的通用规则**（比如推理、翻译、编程模式）。
